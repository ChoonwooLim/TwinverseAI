# Twinverse Platform — 프로젝트 비전

> 이 문서는 모든 AI 파트너가 작업 전 반드시 참조하는 최상위 방침 문서입니다.

## 최종 목표

**미래 인간과 AI가 공존하는 최고의 생태계를 구축한다.**

세계 어떤 플랫폼보다 앞서는 기술력과 사용자 경험으로, 가상과 현실의 경계를 허무는 차세대 플랫폼을 만든다. 타협 없이, 항상 최고 수준을 추구한다.

---

## 핵심 방침 (5대 원칙)

### 1. 세계 최고 품질 — 절대 기준
- 모든 구현은 "세계 최고 수준인가?"를 기준으로 판단한다
- 경쟁 상대는 특정 서비스가 아닌 **전 세계 모든 플랫폼**이다
- AI 파트너는 단순 코딩 도구가 아닌 **세계 최고의 전문가**로서 작업한다
- 기능 하나, UI 하나도 "이것이 세계 최고인가?" 자문 후 실행한다

### 2. Twinverse Platform = 최종 제품
- twinverse.org에 정의된 **Twinverse Platform**이 우리가 만드는 최종 제품이다
- UE5 기반 포토리얼리스틱 가상 공간 플랫폼
- 4K~16K / 120fps 렌더링, 10만 동시접속, 멀티모달 AI 통합
- **목표 규모**: 직접 사용자 2,500만명, 간접 사용자 1억명 (2034년)
- **매출 목표**: 4,770억원 매출, 873억원 순이익 (2034년)

### 3. TwinverseDesk = 플랫폼의 핵심
- TwinverseDesk는 Twinverse Platform의 **핵심 진입점이자 작업 공간**이다
- UE5 C++ 기반 Pixel Streaming으로 브라우저에서 접근 가능한 가상 데스크탑
- AI 1인 기업 시스템의 실행 환경: 50명의 AI 전문가 에이전트가 이곳에서 작동
- **AI 1인 기업 목표**: 12만 기업, 평균 매출 3억원, 총 2,780억원 (2034년)

### 4. 최신 최고 기술 즉시 채택
- 새로운 기술이 나오면 즉시 평가하고, 우수하면 즉시 채택한다
- 기술 부채를 두려워하지 않고 **항상 최전선**에 선다
- 핵심 기술 스택:
  - **렌더링**: UE5 (Nanite, Lumen, Ray Tracing, World Partition)
  - **AI**: GPT-4o, Claude Opus, Gemini, DALL-E 3, Sora, Veo 3
  - **스트리밍**: Pixel Streaming 2 (WebRTC, NVENC)
  - **공간 컴퓨팅**: 디지털 트윈, 물리 시뮬레이션
  - **인프라**: Edge-Cloud 하이브리드, 컨테이너 오케스트레이션
  - **보안**: Zero-Trust, 블록체인 기반 디지털 자산

### 5. 통합 청사진 기반 실행
- 모든 작업은 이 비전 문서와 정합성을 유지하며 수행한다
- 개별 기능 개발도 항상 전체 플랫폼 관점에서 판단한다
- /ultraplan을 활용하여 대규모 작업의 청사진을 수립한 후 실행한다

| **Phase 6** | 플랫폼 경제 | 디지털 자산 마켓, 크리에이터 수익화 |

## AI 파트너 행동 지침

이 프로젝트에서 AI는 단순 도구가 아닌 **세계 최고의 전문가 파트너**이다.

1. **항상 최고 수준을 목표로 한다** — "충분히 좋은" 것은 없다. "세계 최고"만 있다.
2. **최신 기술을 적극 제안한다** — 더 나은 방법이 있으면 주저하지 말고 제안한다.
3. **전체 비전을 항상 인식한다** — 작은 기능도 Twinverse Platform 전체 맥락에서 판단한다.
4. **품질에 타협하지 않는다** — 빠른 구현보다 올바른 구현을 우선한다.
5. **경쟁력을 유지한다** — 구현 시 "세계 어느 사이트보다 뛰어난가?" 자문한다.
