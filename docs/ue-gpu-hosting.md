# GPU 클라우드 호스팅 계획

> 50명+ 동시접속을 위한 GPU 서버 인프라 전략

## 현재 구조

```
현재: 로컬 GPU PC 1대 (RTX GPU)
├── 최대 동시접속: 3명
├── Cloudflare Tunnel로 외부 접근
├── 장점: 무료, 간단
└── 한계: 스케일링 불가, 단일 장애점
```

## 대상 규모별 전략

### Tier 1: 1~5명 (현재)
- **인프라**: 로컬 GPU PC + Cloudflare Tunnel
- **월 비용**: 전기요금만 (~$50)
- **적합 시점**: 개발/테스트, 소규모 데모

### Tier 2: 5~20명
- **인프라**: 클라우드 GPU 서버 1~3대 (상시 운영)
- **추천**: AWS G5.xlarge 또는 CoreWeave RTX A4000
- **월 비용**: $700~$2,100
- **적합 시점**: 초기 서비스 런칭

### Tier 3: 20~50명
- **인프라**: GPU 서버 풀 + 오토스케일링
- **추천**: AWS ECS + G5 Fleet 또는 CoreWeave K8s
- **월 비용**: $2,100~$7,200
- **적합 시점**: 정식 서비스

### Tier 4: 50~200명
- **인프라**: 멀티 리전 GPU 클러스터
- **추천**: AWS GameLift + G5 또는 전용 GPU 클라우드
- **월 비용**: $7,200~$36,000
- **적합 시점**: 대규모 서비스

## 클라우드 서비스 비교

| 서비스 | GPU 타입 | 시간당 비용 | PS2 적합도 | 오토스케일링 |
|--------|---------|-----------|-----------|------------|
| **AWS G5.xlarge** | NVIDIA A10G | ~$1.0/hr | 높음 | GameLift / ECS |
| **AWS G4dn.xlarge** | NVIDIA T4 | ~$0.53/hr | 중간 | GameLift / ECS |
| **CoreWeave** | RTX A4000 | ~$0.61/hr | 매우 높음 | K8s HPA |
| **CoreWeave** | RTX A5000 | ~$0.77/hr | 매우 높음 | K8s HPA |
| **Azure NVasv4** | AMD Radeon MI25 | ~$0.90/hr | 중간 | PlayFab MPS |
| **Google GKE** | NVIDIA T4 | ~$0.35/hr | 높음 | Agones + HPA |

### 추천

1. **최적 가성비**: CoreWeave RTX A4000 ($0.61/hr)
2. **안정성/생태계**: AWS G5.xlarge ($1.0/hr)
3. **저비용 테스트**: Google GKE T4 ($0.35/hr)

### AMI 이미지 구성

GPU 인스턴스에 필요한 소프트웨어:

```
Windows Server 2022 + NVIDIA Driver
├── TwinverseDesk.exe (패키지 빌드)
├── Wilbur Signaling Server (Node.js)
├── GPU Server Backend (Python/FastAPI)
└── Cloudflare Tunnel 또는 직접 포트 노출
```

## 비용 최적화 전략

| 전략 | 절감 효과 | 설명 |
|------|----------|------|
| **유휴 인스턴스 자동 종료** | ~40% | heartbeat 없으면 5분 후 종료 |
| **예약 인스턴스 (1년)** | ~30% | AWS Reserved Instance |
| **스팟 인스턴스** | ~60% | 중단 가능성 있음 (백업 필요) |
| **해상도 최적화** | ~20% | 720p 기본, 1080p 옵션 |
| **시간대별 스케일링** | ~25% | 비피크 시간 인스턴스 축소 |

## 마이그레이션 계획

### Phase 1: 하이브리드 (현재 → 단기)
- 로컬 GPU PC 유지 + 클라우드 1~2대 추가
- 로컬이 가득 차면 클라우드로 오버플로우

### Phase 2: 클라우드 전환 (중기)
- 모든 인스턴스를 클라우드로 이전
- 오토스케일링 활성화
- 로컬 PC는 개발/테스트 전용

### Phase 3: 멀티 리전 (장기)
- 아시아, 미주, 유럽 리전 배포
- 유저 위치 기반 가장 가까운 리전 자동 선택
