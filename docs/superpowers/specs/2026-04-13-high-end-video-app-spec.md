# 하이엔드 AI 영상 자동화 스튜디오 (가칭: Twinverse Video Studio) 기획서

## 1. 프로젝트 개요 (Overview)
기존 매크로 기반 확장 프로그램(예: TubeFlow, TubeFactory)의 한계(브라우저 종속성, 잦은 오류, 일관성 부족, 낮은 프레임 품질)를 극복하고, **기업 및 전문 크리에이터를 위한 최상위(High-end) 급 AI 숏폼/롱폼 영상 자동 생성 플랫폼**을 구축합니다.

### 1-1. 핵심 비전
- **Zero-Click Pipeline:** 대본 기획부터 영상 렌더링, 유튜브 업로드까지 단일 플랫폼에서 완벽하게 동작하는 자동화.
- **Pro-Level Quality:** 뚝뚝 끊기는 슬라이드쇼가 아닌, 실제 영화나 고품질 애니메이션 수준의 일관된 모션과 화면 전환.
- **Studio-Grade UI/UX:** 직관적이면서도 전문가의 미세 조정이 가능한 노드(Node) 및 타임라인 하이브리드 UI 제공.

## 2. 기존 툴(TubeFlow 등)과의 하이엔드 차별화 포인트 (Hybrid Architecture)

| 구분 | 기존 방식 (TubeFlow/Factory 방식) | 향상된 하이엔드 방식 (본 기획) |
| :--- | :--- | :--- |
| **작동 환경** | 단일 웹페이지(구글랩스) 매크로 종속 | **하이브리드 구조 (SaaS 대시보드 + 크롬 익스텐션 봇 연동)** |
| **생성 비용 (Token)** | 구글 무제한/무료 토큰 이용 (장점) | **구글 Flow (ImageFX/VideoFX) 세션/토큰 유지 및 우회 활용 (비용 절감 극대화)** |
| **캐릭터 일관성** | 프롬프트 키워드/이름 강제 주입 (가끔 얼굴 바뀜) | **IP-Adapter / 전용 LoRA 모델 학습**으로 완벽한 인물 일치 |
| **모션 (Video)** | 외부 툴에서 수동 변환 | **익스텐션을 통한 구글 VideoFX(Veo) 자동화 및 외부 API 매끄러운 연동** |
| **타임라인 편집** | 가벼운 싱크 맞춤 | **웹 기반 Pro 타임라인 (입모양 동기화, 트랜지션 내장)** |

## 3. 핵심 기능 명세 (Core Features)

### 3-1. 지능형 뇌: Multi-Agent 기획 엔진
*   **유튜브 트렌드 해커:** 타겟 레퍼런스 URL 입력 시, 조회수 급상승 원인(시각적 훅, 오디오 비트, 스크립트 구조)을 역설계(Reverse Engineering)하여 새로운 스크립트 초안 작성.
*   **프롬프트 엔지니어 에이전트:** 스크립트를 분석해 가장 역동적인 화면 구도를 계산하고, Flux.1 / Midjourney 급의 하이엔드 프롬프트를 자동 생성(카메라 앵글, 조명, 렌즈 값 포함).

### 3-2. 시각적 기획: Cut별 스마트 스토리보드 캔버스 (Smart Storyboarding)
*   **영화 감독급 컷 분할 (Auto Cut-Splitting):** 작성된 대본을 분석하여, 호흡과 내러티브 흐름에 맞춰 전체 영상을 다수의 컷(Scene)으로 스마트하게 분할합니다.
*   **사전 시각화 포스터 (Pre-Visualization):** 무거운 고화질 영상으로 구워내기 전, 각 컷별로 **'빠른 저해상도 이미지(Fast SDXL/Flux-Schnell 등)'를 즉시 생성**하여 스토리보드 형태로 전면에 시각화해 줍니다.
*   **보드 위 직관적 편집 (Drag & Drop):** 사용자는 타일 형태로 펼쳐진 컷들을 보면서 텍스트 대본, 카메라 앵글(Close-up, Pan Right 등), 화면 무드 등을 한눈에 파악하고 만화책을 보듯 편안하게 씬의 순서를 바꾸거나 수정할 수 있습니다.

### 3-3. 완벽한 일관성: Hyper-Consistent 비주얼 엔진
*   **디지털 휴먼 에셋 (Face Lock):** 사용자가 캐릭터 얼굴 사진 1~2장만 올리면, AI가 이를 학습(IP-Adapter FaceID)하여 슬픔, 기쁨, 달리는 모습 등 어떤 씬에서도 **주인공의 얼굴과 옷차림을 100% 고정**시킵니다.
*   **스타일 락 (Style Lock):** '픽사 애니메이션', '사이버펑크', '지브리 스튜디오' 등 화풍이 씬마다 변하는 것을 막기 위해 전역 글로벌 스타일 필터를 적용합니다.

### 3-4. 익스텐션 기반 비디오(Motion) 및 오디오 파이프라인
*   **Google Flow(VideoFX) 비디오 생성 자동화:** 브라우저 익스텐션이 사용자의 로그인 세션과 토큰을 사용하여 구글의 최신 Video AI (Veo 등) 화면에서 텍스트 또는 이미지를 비디오로 자동 변환하고 다운로드합니다. (API 비용 대폭 절감)
*   **감정 인식 TTS (ElevenLabs / AI 커버 연동):** 대본의 문맥을 파악하여 소리지르는 구간, 속삭이는 구간을 자동으로 파악하고 음성의 톤/속도를 조절합니다.
*   **Lip-Sync (입모양 동기화):** 생성된 AI 캐릭터의 입모양을 한국어/영어 음성 발음에 맞게 정확하게 애니메이팅 합니다. (비디오 품질의 끝판왕 기능)
*   **자동 B-Roll & 사운드 이펙트:** 대본에서 '폭발음', '빗소리' 등의 키워드를 감지하여 적절한 배경 효과음과 시각적 트랜지션을 자동 삽입합니다.

### 3-5. 프로페셔널 웹 타임라인 
*   **브라우저 렌더링 엔진:** WebGL 및 WebCodecs API를 활용해 프리미어 프로 수준의 다중 트랙(비디오, 오디오, 자막) 편집 환경을 웹에 제공합니다.
*   **원클릭 수정:** 마음에 들지 않는 특정 씬만 클릭하여 "이 부분 비를 내리게 해줘"라고 텍스트로 치면 해당 부분만 즉시 재수정(Inpainting) 됩니다.

## 4. 시스템 아키텍처 및 기술 스택 (Tech Stack)

*   **Frontend (사용자 인터페이스):**
    *   `Next.js 16` 및 `React 19`: 극한의 퍼포먼스와 서버사이드 렌더링.
    *   `Tailwind v4` & `Framer Motion`: 하이엔드 글래스모피즘(Glassmorphism) 및 반응형 유체 애니메이션 UI.
*   **Backend & Extension 연동 아키텍처:**
    *   `UI Dashboard` (웹) <-> `WebSocket / Relay` <-> `Chrome Extension` (생성 워커).
    *   익스텐션은 백그라운드에서 구글 Flow 봇 역할을 유지하며 비디오 및 이미지를 생성, 결과물을 SaaS 대시보드로 자동 업로드합니다.
*   **AI Models & APIs (Hybrid):**
    *   **Text/Script:** OpenAI GPT-4o, Anthropic Claude 3.5 Sonnet
    *   **Extension Worker (무료 토큰):** ImageFX (이미지), VideoFX / Veo (영상) 세션 활용
    *   **Premium Video (선택적 API):** Runway Gen-3 Alpha, Kling AI API (고사양 렌더링용)
    *   **Voice/Audio:** ElevenLabs API, Suno API (배경음악)
*   **Database:** `PostgreSQL` (유저 상태 및 프로젝트 데이터), `Supabase` (실시간 상태 동기화).

## 5. 비즈니스 수익화 모델 (Monetization)

*   **크레딧/토큰 기반 종량제 결제:** AI 모델 호출 비용이 크므로, 구독과 크레딧을 혼합한 모델.
*   **티어(Tier) 설계:**
    1.  **Starter (초보/취미):** 텍스트 -> 이미지 메인, 숏폼 월 10개 생성 제한.
    2.  **Pro (주력 상품 - 월 4~5만원 대):** 타임라인 편집기 해제, 립싱크 가능, Face Lock (캐릭터 일관성) 지원. 비디오 렌더링 우선권.
    3.  **Agency (기업용 - 연간 계약):** 커스텀 AI 모델 서버 할당, 협업(팀원 초대) 기능, 상업용 라이선스 완벽 보장.
